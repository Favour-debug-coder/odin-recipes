# Savory Bites Restaurant

## Description
This is a clean, modern HTML website built as my first project for **The Odin Project** curriculum. Instead of a standard recipe page, I designed a landing page for **Savory Bites Restaurant**. It features an introduction to the restaurant, a visual preview, a fully categorized menu, and links to stay connected.

## Features
* **Structured Menu:** Uses nested HTML lists to separate Starters, Main Dishes (including our signature Jollof Rice!), Drinks, and Desserts.
* **Media Integration:** Displays a high-quality image placeholder representing the restaurant's cozy environment.
* **Navigation:** Includes anchor tags that link out to external restaurant resources.

## Skills Demonstrated
* Writing clean, semantic HTML5 structural elements.
* Implementing advanced nested list structures (`<ul>` inside `<li>`) for clean data presentation.
* Working with local file paths and image attributes.
* Using Git and GitHub for version control, repository cloning, and project
